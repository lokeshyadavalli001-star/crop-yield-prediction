## 📌 Problem Statement
Agriculture plays a vital role in food production and economic growth. However, accurately estimating crop yield is challenging because it depends on multiple factors such as rainfall, temperature, and other environmental conditions.
Traditional yield estimation methods often rely on manual observations and historical trends, which may not always provide accurate predictions. The objective of this project is to develop a Machine Learning-based system that can predict crop yield using agricultural and environmental data.

By leveraging supervised machine learning algorithms such as Linear Regression and Random Forest Regressor, this system aims to provide data-driven yield predictions that can assist farmers, researchers, and policymakers in making informed agricultural decisions.

---

## 🎯 Objective

Accurate crop yield prediction can help farmers and agricultural stakeholders make informed decisions regarding resource allocation, crop planning, and productivity optimization. Machine learning approaches such as Random Forest are widely used for agricultural prediction tasks because they can capture complex relationships between environmental variables and yield outcomes.

## 📊 Machine Learning Models

### Baseline Model

* Linear Regression

### Final Model

* Random Forest Regressor

The Linear Regression model was used as a baseline for comparison, while Random Forest was selected as the final model due to its ability to model non-linear relationships and interactions between agricultural features. Random Forest is commonly used in crop-yield prediction applications and often outperforms simple linear models in such settings.
